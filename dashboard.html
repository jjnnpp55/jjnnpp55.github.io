Employer Dashboard Technical overview of the Quiz Universe project.

Project Scope

  • Built a 720 page interactive quiz website with 7,200 questions.
  • Organized into 8 knowledge categories with 6 sub-topics each.
  • Core navigation from subtopics to each quiz
  • Includes a timer, a JSON-driven dictionary quiz, and a "Spin the Wheel" toy.
  • Integrated affiliate program links for monetization.
  • Plans to expand with more categories and video

Frontend & Styling

  • HTML5, CSS Grid & Flexbox were used for a responsive design.
  • Used a mix of Inline, Embedded, and External CSS for styling.
  • Inserted targeted Meta Tags to enhance search engine attention and improve discoverability.
  • Implemented a clean Column-Based Layout to optimize content readability and create a balanced user interface.
