TP-Link Beginner Quiz 3
1. What is 'Address Reservation' (DHCP)?
Assigning a permanent IP address to a specific device
2. What is 'MAC Filtering'?
Allowing or Blocking devices based on their hardware address
3. What is 'Beamforming'?
A feature that allows you to control internet usage for specific devices or users
4. What is 'MU-MIMO'?
A technology that allows devices to connect to the internet wirelessly
5. What is 'AP Mode' (Access Point)?
A unique name that identifies a wireless network
6. What is 'IPTV/VLAN' setting for?
A security protocol that encrypts wireless communication
7. What is 'USB Sharing'?
A feature that allows you to prioritize certain types of network traffic
8. What is 'DDNS'?
A feature that allows you to share files and resources between devices on your network
9. What is 'Schedule Reboot'?
A feature that allows you to create a separate network for guests
10. What is 'SPI Firewall'?
A feature that allows you to block unwanted websites or content
